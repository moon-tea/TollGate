TollGate
========

Java - A Toll Gate program that can add customers and let them drive through tolls 

```c++
+===========+===================================+===========================+===============================+=======================================================================+
|Input	    |Output							    |Designates				    |Example Input				    |Example Output                                                         |
+===========+===================================+===========================+===============================+=======================================================================+
|S			|Toll Gate Program started...	    |The start of the file	    |S							    |Toll Gate Program started...                                           |
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|
|U s+ s+    |Name: name user ID: userID 		|Create a customer account	|U MonteCNichols dukeOfMilan	|Name: MonteCNichols user ID: dukeOfMilan                               |
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|
|A s+ s+    |Vehicle Added to user s+: Plate 	|Add Vehicles				|A dukeOfMilan PK1368		    |Vehicle Added to user dukeOfMilan: Plate #: PK1368 TollTag #: 000001   |
|		    |#: s+ TollTag #: 000000			|		                    |                               |                                                                       |
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|
|D cc_## s+ |Plate: ssssss cost: $0.00			|A zipcash user				|D GB_01 QZ345G 				|Plate #: QZ345G drove through GB_01                                    |
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|
|R cc_## s+ |Customer ID: s+ cost: $0.00		|A tolltag user				|R GB_01 dukeOfMilan 			|Customer: dukeOfMilan drove through GB_01                              |
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|
|T 			|TollTag User: s+ owes: $0.00       |Print Toll Value           |T                              |TollTag User: dukeOfMilan owes: $1.00 ZipCash Plate: QZ345G owes: $1.50|
|			|ZipCash Plate: s+ owes: $0.00		|	                        |                               |                                                                       |										
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|
|E			|Toll Gate Program ended...			|Exit the program			|E								Toll Gate Program ended...                                              |
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|
|#			|									|Commented out inputs		|# Next line adds a new customer|	                                                                    |
|-----------|-----------------------------------|---------------------------|-------------------------------|-----------------------------------------------------------------------|				
|           |			                        |
+===========+===================================+
|Symbol Key	|			                        |
+===========+===================================+
|c			|char			                    |
|-----------|-----------------------------------|
|#			|decimal number			            |
|-----------|-----------------------------------|
|s			|string			                    |
|-----------|-----------------------------------|
|0			|leading or trailing 0              |
|-----------|-----------------------------------|
```
